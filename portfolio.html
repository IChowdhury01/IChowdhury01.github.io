ItsMe: Attachable Smart Lock

ItsMe is an Android app that controls an attachable smart lock IoT device remotely through a server hosted on a Raspberry Pi. The goal of this project is to create a more versatile form of lock that can be freely attached onto deadbolt doorlocks and then controlled remotely via a smartphone application. The lock's design is open-source and thus modifiable to the owner's preferences, 3D-printable from standard materials, and requires low-cost installation without home renovation.
